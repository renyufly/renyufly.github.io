​ edX链接： CS50’s Web Programming with Python and JavaScript | Harvard University

​ B站含中文字幕完整版：Harvard CS50’s Web Programming with Python and JavaScript_bilibili

👉 课程介绍： 本课程从CS50x开始，更深入地探讨使用 Django、React 和 Bootstrap 等框架使用 Python、JavaScript 和 SQL 设计和实施 Web 应用程序。主题包括数据库设计、可伸缩性、安全性和用户体验。通过实践项目，学生学习编写和使用 API、创建交互式 UI 以及利用 GitHub 和 Heroku 等云服务。到学期结束时，学生将掌握原理、语言和工具方面的知识和经验，使他们能够在 Internet 上设计和部署应用程序。

👉 课程内容： HTML, CSS, Git, Python, Django, SQL、模型和迁移, JavaScript, 用户界面, 测试、CI/CD, 可扩展性和安全性
