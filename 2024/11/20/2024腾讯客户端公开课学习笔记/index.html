课程安排：

将UE5.4 源码编译并打包到安卓

源码编译

Epic申请绑定github账号，

下载Android Studio(最新版即可），在SDK Platform与SDK Tool根据UE官网要求选择下载对应版本 ：Set Up Android SDK, NDK, and Android Studio using Turnkey for Unreal Engine | Unreal Engine 5.4 Documentation

需要电脑中配有Java环境 （且Android Studio2024版要求JDK17）。

对下载下来的源码，进入文件夹，运行脚本文件：

用 Visual Studio 打开 UE5.sln，右键UE5， 右键 “Build” 开始编译。

首次编译时间：16：30 - 22：15 总用时尽 6 h （笔记本配置为11th Intel i7-11800H ， 16G内存， RTX3060Laptop）

编译完成之后找到地址 \Engine\Binaries\Win64\UnrealEditor.exe 并启动。之后选择 “Game” → 第三人称 游戏 ，

项目设置里 目标平台选择 移动平台 （其实桌面和移动没啥区别，只是选移动，之后打包快些），质量预设选择 可缩放，项目名称为全英文，其他光线追踪、初学者内容包都不勾选。

创建完成之后会自动打开VS2022，在Solution Explorer里会显示刚创建的项目工程（在Games文件夹下），右键项目名称，选择 “Build”

（耗时 2 min）

之后再打开 \Engine\Binaries\Win64\UnrealEditor.exe， 选择项目。 （注意是从源码编译出来的目录里的Editor.exe）

一些设置：在editor preference里all settings → Asset Editor Open Location 选择 Main Window来使得只创建新标签页，不打开新窗口。

项目设置

Project settings → Platform → Android ：将 APK Packaging 和 Google Play Services 启动配置

  • ​ Target SDK Version (26=8.0.0, 27=8.1.0, 28=9, 29=10, 30=11, 31=12) 根据对应的安卓平台设置 （这里Android14就写为 34）
  • ​ 勾选：Package game data inside .apk?
  • ​ 勾选：Allow large OBB files.
  • ​ 勾选：Allow patch OBB file.
  • ​ 勾选：App Bundles / Generate bundle (AAB)

Project settings → Platform → Android SDK ：

  • Location of Android SDK (the directory usually contains’android-sdk-) ：设置为 Android / SDK （这是下Android studio时下载的SDK的目录）
  • Location of Android NDK (the directory usually contains ‘android-ndk-) ：设置为 Android / SDK / ndk / 25.1 （版本号根据UE官网）
  • Location of JAVA (the directory usually contains ‘jdk’) ： 设置为 XPAN / JDK / 1.17 （这是自己配置 %Java_Home% 的地址）
  • SDK API Level (specific version, ‘latest’, or ‘matchndk’ - see tooltip) ： 设置为 “matchndk”
  • NDK API Level (specific version or ‘latest’ - see tooltip) ： 设置为 “android-25” （后面的数字为自己设置的SDK版本）

打包安卓

首先可安装 anlink 便于手机投屏到电脑 （也可使用 scrcpy ）。

打开项目后，在 右上角的 settings → Preview Platform 选择为 Android ，在上方 Play 按钮 旁边 的 Platforms 选择 Android → Package Project。 下方的Flavor selection 根据需要选择对应的纹理压缩格式。 可见 Android Development Basics for Unreal Engine | Unreal Engine 5.4 Documentation

等到控制台显示 BUILD SUCCESS后，打开自己选择的打包文件夹，选择 Install_hw1-arm64.bat(安装apk脚本) 与 Uninstall_hw1-arm64.bat (卸载脚本)。 之后即可在手机上打开软件。

注意：安卓手机要打开 “开发者模式”。
